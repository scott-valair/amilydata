# Amily Data Analysis Website

This is a responsive one-page website for Amily Data Analysis.

## Files

- `index.html` — website structure and content
- `styles.css` — responsive design, layout, animation, and visual polish
- `script.js` — mobile menu, active navigation, animated counters, service card details, carousel controls, chart refresh, request builder, copy buttons, and contact form behavior
- `assets/amily-data-logo.png` — your logo
- `assets/scott-valair-portrait.jpg` — your portrait
- `assets/favicon.png` — browser tab icon
- `client-intake-template.csv` — simple questions to ask new clients
- `IMPLEMENTATION.md` — publishing instructions

## Preview locally

1. Unzip the website folder.
2. Open `index.html` in a browser.
3. Resize the browser to check desktop, tablet, and mobile layouts.

## Contact form note

The contact form uses `mailto:`. It opens the visitor’s email app with the message filled in. The message is not sent until the visitor sends it from their own email app.

For a real web form, connect the form to your host’s form handler or a form service.

## Quick edits

To change your email, search for:

`scott.valair@amilydata.com`

Edit it in both `index.html` and `script.js`.

To change service descriptions, edit the service cards in `index.html` and the `serviceContent` section in `script.js`.

To replace sample dashboards, edit the Sample Work section in `index.html`.
